# Frontend de Movie-API

## Descripción del proyecto
Este proyecto es el frontend de la API Movie-API. Permite visualizar las películas disponibles y crear reseñas para cada una de ellas.

## Tecnologías utilizadas
- React
- JavaScript (JS)
- CSS
- HTML
